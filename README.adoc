== Usage

To use `SNAPSHOT` version.

[xml]
----
<repositories>
    <repository>
        <id>jfrog-oss-snapshot</id>
        <name>JFrog OSS Snapshots</name>
        <url>https://oss.jfrog.org/libs-snapshot</url>
        <releases>
            <enabled>false</enabled>
        </releases>
        <snapshots>
            <enabled>true</enabled>
        </snapshots>
    </repository>
</repositories>
----
